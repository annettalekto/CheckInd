Программа проверки индикаторов

Программа предназначена для настройки и проверки:

  • индикатора скорости цифрового (ЦАКТ.467845.001);
  • индикатора дополнительного (ЦАКТ.467845.006 всех исполнений);
  • модуля реле (МР-3ПА ЦАКТ.468361.013 и МП-3П ЦАКТ.468361.009).

Проверяемая плата должна быть подключена к плате интерфейсной, плата интерфейсная к компьютеру. Настоящая программа осуществляет обмен данными с программой платы интерфейсной и вывод на экран информации об активных элементах (включенные сегменты, нажатые кнопки, установленные выходы реле) и ошибках.

После запуска программы на экране появляется окно программы в соответствии с рисунком 1. В нижней части окна расположены вкладки для проверки:

  • основного индикатора;
  • дополнительного индикатора и кнопок;
  • блока реле;
  • информации о программе.

Рисунок 1

Проверка индикаторов

Индикаторы отображаются в виде набора разрядов (рисунки 1 и 2), состоящих из отдельных сегментов. Включенному сегменту индикатора соответствует зеленый цвет. Если в процессе обмена по шине I2C с одним из регистров произошла ошибка, то сегмент, соответствующий данному регистру отображается красным.

Индикаторы можно тестировать в ручном и автоматическом режимах.

В ручном режиме пользователь может включать и выключать отдельные сегменты индикатора, нажимая мышью на экранном изображении индикатора. Цвет сегмента должен измениться на экране и на плате должен быть подсвечен соответствующий сегмент. Кнопка «Сброс» выключает все сегменты.

Для включения автоматического режима нажмите кнопку «Старт», для выключения «Стоп». В автоматическом режиме программа синхронно переключает по одному сегменту каждого индикатора. Частоту переключения можно выбрать из выпадающего списка рядом с кнопкой «Старт». По умолчанию частота переключения сегментов 1 Гц.

При ошибке связи с платой интерфейсной выводится сообщение.

Проверка кнопок

На вкладке «Дополнительный индикатор» программа производит периодический опрос кнопок. Нажатые на плате кнопки отображаются на экране голубым цветом. В случае ошибки обмена по шине I2C с регистром состояния кнопок, все кнопки будут выведены красным цветом.

Рисунок 2

Проверка реле

Блок реле можно тестировать в ручном и автоматическом режимах. В ручном режиме пользователь может изменять значения отдельных выходов, нажимая мышью на экранном изображении соответствующего бита (0 – 4) для его инверсии. Установленному биту соответствует голубой цвет. В автоматическом режиме значения двоичных выходов изменяются автоматически (по схеме «бегущая единица»). Автоматический режим включается и выключается кнопками «Старт»/«Стоп», частота выбирается из выпадающего списка. При ошибке связи с платой интерфейсной выводится сообщение.

Рисунок 3

Информация

Вкладка «Информация» отображает:

  • версию настоящей программы;
  • версию программы платы интерфейсной;
  • статус соединения (виртуальный COM-порт) с платой интерфейсной;
  • доступные для подключения COM-порты.

Рисунок 4

Если на вкладке «Информация» номер версии программы платы интерфейсной отображается, то плата интерфейсная подключена верно и обмен данными осуществляется корректно. Если программа не смогла определить COM-порт, то вместо «Соединение установлено: COM», будет отображена ошибка. В этом случае выберите нужный COM-порт из выподающего списка «Подключить COM-порт».
